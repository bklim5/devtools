Current Platform interface (src/lib/platform/index.ts):
```typescript
export interface Platform {
  clipboard: { writeText(text: string): Promise<void>; readText(): Promise<string> };
  store: Store;
}
export const platform: Platform = {
  get clipboard() { return active.clipboard; },
  get store() { return active.store; },
};
```

tauri.ts is the ONLY @tauri-apps importer. Add window control via @tauri-apps/api/window
(getCurrentWindow) and shortcuts via @tauri-apps/plugin-global-shortcut.

The new interface shape to add (per RESEARCH Pattern 1):
```typescript
window: {
  show(): Promise<void>;
  setFocus(): Promise<void>;
  unminimize(): Promise<void>;
  minimize(): Promise<void>;
  isVisible(): Promise<boolean>;
};
nativeShortcut: {
  register(accelerator: string, handler: () => void): Promise<void>;
  unregister(accelerator: string): Promise<void>;
  isRegistered(accelerator: string): Promise<boolean>;
};
```

Existing platform.test.ts builds a `Platform` stub object literal inline (clipboard+store) —
those literals MUST be widened to include the two new caps or `tsc` fails.

    ```typescript
      // macOS focus regression (Pitfall 1 / issue #12834): order matters. The shell
      // summon flow (Plan 03) calls unminimize() → show() → setFocus() in that order.
      async show() { await getCurrentWindow().show(); },
      async setFocus() { await getCurrentWindow().setFocus(); },
      async unminimize() { await getCurrentWindow().unminimize(); },
      async minimize() { await getCurrentWindow().minimize(); },
      async isVisible() { return getCurrentWindow().isVisible(); },
    },
    nativeShortcut: {
      // Filter to Pressed so the handler fires once per chord, not on key-up too.
      register: (acc, h) => register(acc, (e) => { if (e.state === "Pressed") h(); }),
      unregister: (acc) => unregister(acc),
      isRegistered: (acc) => isRegistered(acc),
    },
    ```
