| Modal DOM ↔ background | aria-modal promises the background is inert; a broken trap would let focus/interaction leak out |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-22-01 | Tampering | `#/settings/license` deep-link | mitigate | The deep-link only calls `openSettings("license")` against a fixed literal pane id; the hash is never parsed into a dynamic pane/route target (SettingsDeepLink hardcodes "license") |
| T-22-02 | Information disclosure | Settings modal hosting LicenseSettings | accept | The License pane reuses the unchanged Phase-21 surface; the masked key/email path (statusDetail, Rust-side masking) is untouched — no new key material crosses to JS |

| T-22-04 | Elevation of privilege | shell modal reachable by everyone (D-S10) | accept | Intentional — Settings opens for unlicensed users; the License pane's own entitlement gating is unchanged; the shell adds no privileged action |
